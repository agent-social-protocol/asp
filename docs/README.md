# Public Docs

This directory contains the public, protocol-facing documentation surface for
the `asp` repo.

## Kept Public

- `asp-spec-01.md` — protocol specification
- `minimum-compliant-node.md` — minimum compliant node checklist
- `identity-and-discovery-model.md` — current identity/discovery model
- `runtime-surface-capabilities.md` — reference agent-facing CLI/MCP surface contract
- `asp-whitepaper.md` — public narrative and positioning document

## Kept Private

Working notes, future-change proposals, maintenance checklists, and
application-facing conventions live in the private `platform` repo.

If a document is primarily about product positioning, hosted onboarding,
private implementation planning, internal maintenance process, or
application-specific workflows, it does not belong in this public repo.
